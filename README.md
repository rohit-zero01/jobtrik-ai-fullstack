# 🤖 AI Interview Preparation Platform

A full-stack AI-powered web application that generates personalized interview strategies based on job descriptions and user profiles.

---

## 🚀 Features

* 🔐 JWT Authentication (Login / Register / Logout)
* 📄 Resume Upload & Analysis
* 🧠 AI-powered Interview Strategy Generation (Google Gemini API)
* 📊 Match Score Calculation
* 🗂️ Interview Report History
* ⚡ Clean and responsive UI

---

## 🛠️ Tech Stack

### Frontend

* React (Vite)
* SCSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas

### AI Integration

* Google Gemini API

---

## 🔐 Environment Variables

Create a `.env` file in backend:

MONGO_URI=your_mongo_uri
GOOGLE_GENAI_API_KEY=your_api_key
JWT_SECRET=your_secret

---

## ▶️ Run Locally

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm run dev

---

## 🌐 Deployment

* Backend: Render
* Frontend: Render / Vercel
* Database: MongoDB Atlas
